# Svendsén Photography

Välkommen till **Svendsén Photography**, en kombinerad fotografiportfölj och CV-hemsida. Denna webbplats är byggd från grunden med fokus på att presentera professionella fotografitjänster samt visa upp min kompetens som fullstackutvecklare.

## Funktioner
- **Fotografiportfölj**:
  - Kategorier: Bröllop, porträtt, företagsfotografering och familjefoton.
  - Bildkarusell med slumpmässiga bilder.
  - Responsiv design som fungerar på både mobil och dator.
- **CV-sektion**:
  - Information om utbildning, arbetslivserfarenhet och tekniska kompetenser.
  - Möjlighet att ladda ner CV som PDF.
- **Kontaktformulär**:
  - Enkel kontakt via formulär för bokningar och förfrågningar.
- **SEO-optimerad**:
  - Metadata och sociala media-taggar för att förbättra synligheten.

## Teknologier
- **Frontend**: React med React Router och Tailwind CSS.
- **Byggverktyg**: Vite.
- **Hosting**: GitHub Pages.
- **Formulärhantering**: Formspree.io för kontaktformulär.
- **SEO**: React Helmet för att hantera metadata.
