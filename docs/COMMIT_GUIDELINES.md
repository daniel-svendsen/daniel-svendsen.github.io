# Commit Guidelines

Do not add `Co-authored-by`, AI attribution, or tool attribution unless the user explicitly asks.

For trivial commits, a short imperative subject is enough:

```text
Fix gallery modal spacing
```

For non-trivial commits, use:

```text
Short imperative subject

- What changed
- Why it changed
- Risk or user impact
- Tests/checks run
```

Keep the subject concise and specific. Use 2-5 bullets in the body.
